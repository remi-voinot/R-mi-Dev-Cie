Interface web de configuration

Vous pouvez gérer les options de votre bot Discord via une interface web. Ajouter des fonctionnalité sera un jeu d'enfant pour vous !

Sécurité oblige !

Comme votre bot est susceptible de contenir des informations sensibles, nous garantissons sa sécurié en empechant toute intrusion dans les fichier du bot. Les bases de données sont cryptés pour garantir une sécurité maximale.
